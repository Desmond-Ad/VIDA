⚠️ MANUAL STEPS REQUIRED:

1. INSTALL POSTGRESQL:
   - Download from: https://www.postgresql.org/download/windows/
   - Install with default settings
   - Remember the password you set for postgres user

2. CREATE DATABASE:
   - Open pgAdmin (installed with PostgreSQL)
   - Connect to PostgreSQL server
   - Create database named "vida"

3. UPDATE ENVIRONMENT VARIABLES:
   - Create .env file in project root
   - Add: DATABASE_URL=postgresql://postgres:YOUR_PASSWORD@localhost:5432/vida
   - Replace YOUR_PASSWORD with your postgres password

4. RUN SERVER:
   - npm install (to install new dependencies)
   - node server.js

⚠️ IMPORTANT NOTES:

   → Allow Node.js/port 5000 if blocked
